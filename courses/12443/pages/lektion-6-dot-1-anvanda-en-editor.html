Lektion 6.1 - Använda en editor

Vi kommer byta verktyg från kodlabbet till ett mer användbart verktyget. Kodlabbet har fördelen att det är snabbt att komma igång med, men vi tappar en del funktionalitet och kontroll över vad vårt program kan göra.

Den editor som i princip är standard för webbutveckling är Visual Studio Code (VSCode), men det finns många andra bra verktyg. Vill du använda en annan editor än Visual Studio Code (t ex Sublime) är det OK, men jag kommer använda Visual Studio Code i mina exempel.

Om du har en ChromeBook så kan du endast köra on-line versionen av VSCode.

Gå till https://code.visualstudio.com/ och följ instruktionerna för ditt operativsystem.

Online-versionen av VSCode finns på https://vscode.dev/. Lägg ett bokmärke i din browser till den här sidan om du använder den. Du måste även tillåta sidan att komma åt lokal lagring.
