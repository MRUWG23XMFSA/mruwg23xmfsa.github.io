Lektion 6.2 - Skriva kod i VSCode

Ett JavaScript existerar inte bara för sig själv utan scriptet måste köras i browser. (Det här är inte helt sant, men för våra syften stämmer det bra.) För att browsern ska läsa vårt script så läser den först in en html-fil. HTML är det format som en webbsida beskrivs av, mer om detta senare.  Vi behöver en html sida som vårt script får vara en del av. Detta görs med särskilda HTML-taggar. Här är ett exempel på hur ett script som ritar en kvadrat på skärmen ser ut. Det är något längre och kanske lite krångligare än vad vi sett tidigare, men nu tar vi själva full kontroll över vår kod.

<!DOCTYPE html>
<html>
    <head>
        <title>Min första canvas</title>
        <meta charset="UTF-8" >
    </head>
    <body>
       <h1> Här är en html-canvas</h1>
    <canvas id="minCanvas" width="400" height="400"></canvas>
     <script>
      var canvas = document.getElementById("minCanvas");
      var context = canvas.getContext("2d");
      context.strokeStyle ="Gold";
      context.fillStyle="DarkSlateBlue"
      context.lineWidth = 5;
      context.rect(75,75, canvas.width-150 ,canvas.height-100);
      context.stroke();
      context.fill();
     </script> 
    </body>
</html>
