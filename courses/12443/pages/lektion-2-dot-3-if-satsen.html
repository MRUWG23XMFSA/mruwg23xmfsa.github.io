Lektion 2.3 If-satsen

Villkor

Ett vanligt kommando för att kontrollera villkor är if-satsen. Vi ställer t ex frågan om en person är myndig genom att kontrollollera om en person är unde18 år. Om vi vill att vårt program ska ställa en fråga kan vi använda kommandot prompt

Pröva koden nedan och se till att du förstår hur den fungerar. If-satsen följs alltid av ett block, dvs vi måste använda klammer (måsvingar) även om det bara är en rad som ska utförast.

function start() {
    var name = prompt("Hej, vad heter du?");
    var age = prompt("Hur gammal är du?");
    text(50, 150, 20, "Hej " + name + "!", "black");    
    if (age < 18) {
        text(50, 250, 20, "Du är inte myndig" , "black");
    }
    else {
        text(50, 250, 20, "Du är myndig" , "black");
    }
}

jämförelseoperatorer i JavaScript

Om vi vill jämföra om talet a är lika med 3 måste vi JavaScript använda dubbla likhetstecken.

if a == 3 {
    text(50, 150, 20,"a är lika med 3", "blue");
}
else {
    text(50, 150, 20,"a är inte lika med 3", "red");
}
