Lektion 2.2 - Animering

En rörelse på en datorskärm är helt enkelt flera stillbilder ritade efter varandra. Det är ingen skillnad på hur t ex en film eller en tv-bild uppdateras. Om vi byter bild tillräckligt ofta så upplever vår hjärna en rörelse. I simple.js biblioteket finns en färdig funktion som heter update för att rita om innehållet på skärmen. Om vill att vår boll ska röra sig över skärmen så kan vi pröva följande kod

function start() {
    ball = {x: 0, y:0}
}

function update() {
    circle(ball.x, ball.y , 35, "blue");
    ball.x++;
    ball.y++;
}

Om vi vill att spåret efter bollen inte ska synas behöver vi börja uppdateringen med att rensa skärmen och rita allt på nytt igen.

function update() {
    clearScreen()
    circle(ball.x, ball.y , 35, "blue");
    ball.x++;
    ball.y++;
}

Uppdateringshastigheten kan styras via varabeln updatesPerSecond. Denna variabel är till början satt till 30, vilket innebär att skärmen uppdateras 30 gånger per sekund. Du kan ändra den men en väldigt hög uppdateringshastighet kommer vara tungt för datorn.

Man kan stoppa funktionen update genom att kalla på funktionen . Det kan vara användbart när användaren förlorat i ett spel.
