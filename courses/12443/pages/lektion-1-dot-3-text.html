Lektion 1.3 - Text

Att skriva ut text

Vi vill gärna att våra program kan skriva ut text, det kan t ex vara ett svar från en beräkning eller kanske att uppdatera poäng i ett spel. Om vi använder oss av simple.js finns ett enkelt sätt att skriva ut text. Koden nedan bör vara ganska enkel att förstå.

<script src = "https://koda.nu/simple.js">

    text(50, 150, 40, "Hej världen!", "blue");

</script>
