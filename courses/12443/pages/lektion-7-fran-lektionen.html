Lektion 7 - från lektionen

Vi gjorde klart funktionen för att rita pacman, du hittar en lösning här.

Det dök upp en fråga om hur man kunde få slumpartade färger som inte riktigt hanns med.

För att skapa ett slumptal så används Math.random().  Då får man ett decimaltal mellan 0 och 1. Färger anges med värden för rött, grön och blått (RGB) där varje färg ska ha ett värde mellan 0 och 255. För att få detta multiplicerar vi vårt slumptal med 255, Math.random()*255. Då har vi fortfarande ett decimaltal, vi är intresserade av decimalerna utan kan klippa bort dessa med funktionen Math.floor(). Så för att tilldela variablen r ett slumptal mellan 0 och 255 kan vi skriva

let r = Math.floor(Math.random() * 255));

Eftersom detta är något vi antagligen vill göra flera gånger så är det vettigt att skapa en funktion. En egenhet hos t ex fillStyle är att den förväntar sig en textsträng av typen "rgb(24,55,123)", så vår funktion slumpar fram tre tal och bygger sedan en sträng som returneras.

 

<body>
    <canvas id="myCanvas" width="150" height="150"></canvas>
    <script>
        const canvas = document.getElementById("myCanvas");
        const ctx = canvas.getContext("2d");

        function randomRGB(){
            // Slumpa fram tre värden mellan 0 och 256 för varje färg.
            // Måste returnera en sträng för att fungera
            let r = Math.floor(Math.random()*256);
            let g = Math.floor(Math.random()*256);
            let b = Math.floor(Math.random()*256);
            return 'rgb(' + r +','  + g +',' + b + ')'
        }

        ctx.fillStyle = randomRGB();
        ctx.fillRect(10,10,140,140);
        ctx.stroke();
        ctx.fill();

    </script>    
</body>
