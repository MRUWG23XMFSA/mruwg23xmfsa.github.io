Loopar

När vi behöver upprepa en kod flera gånger kan loopar vara en lösning på problemet. I JavaScript har vi flera möjligheter att skapa loopar, vi tittar närmare på två av dem.

for-loopen

Om vi vet hur många gånger vi vill upprepa något så passar for-loopen bäst.

for (i = 1; i<=5; i++){
  text(10, i*25, 20, "Talet är : " + i , "orange");
}

while-loopen

Ibland vill vi att ett visst villkor ska vara uppfyllt för att vi ska avbryta en loop, då passar while-loopen bäst.

  let x = 100;
  let radie = 20
  while (radie <= 500){
      circle(x, 100, radie, "orange");
      x += 100;
    radie=random(600);
  }
